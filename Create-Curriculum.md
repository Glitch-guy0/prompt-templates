You are a curriculum designer. Using the sources I've provided, 
create a structured learning curriculum broken into Modules 
(Module 1, Module 2, Module 3...).

Each module should cover ONE focused topic and be completable 
in 30-minute daily sessions.

---

FORMAT FOR EVERY MODULE:

**Module [N] — [Topic Name]**

🎯 Learning Goal  
One clear sentence on what the learner will be able to do 
after this module.

📖 Concept Overview (10 mins)  
Explain the core idea in simple terms. Use an analogy if possible.

⚡ Mini Exercise (10 mins)  
A small, immediate task the learner can try right now to 
test the concept. Keep it simple and self-contained.

🌍 Real-World Case Study  
A short real-world example or scenario showing this concept 
applied in practice.

🔨 Hands-On Project Task  
One small project step that ties this module's concept into 
a larger running project that builds across all modules.

---

END OF MODULE — Full Review:

📝 Quick Quiz  
3 MCQ questions testing the key ideas from this module.
Include the correct answer and a one-line explanation.

🏗️ Module Project Checkpoint  
A clear deliverable the learner should have built or completed 
by end of this module.

✅ Reflection Checklist  
3–4 yes/no questions the learner self-checks:  
- Can I explain [concept] in my own words?  
- Did I complete the mini exercise?  
- Can I connect this to a real use case?  

---

- Do NOT summarize the source material. BUILD a curriculum from it.
- Each module must have all sections filled — no skipping.
- Modules should flow logically from foundational to advanced.
- The hands-on project should be one continuous project that 
  grows across modules, not separate isolated tasks.
